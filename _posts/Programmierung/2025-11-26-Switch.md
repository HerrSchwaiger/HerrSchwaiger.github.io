Das switch-case-Statement ist eine Kontrollstruktur, mit der du verschiedene Codeteile basierend auf dem Wert einer Variable ausführen kannst. Es ist eine Alternative zu mehreren aufeinanderfolgenden if-else-Anweisungen und macht den Code übersichtlicher, wenn du viele verschiedene Fälle prüfen musst.

## Syntax
```java
switch (ausdruck) {
    case wert1:
        // Code für wert1
        break;
    case wert2:
        // Code für wert2
        break;
    case wert3:
        // Code für wert3
        break;
    default:
        // Code, wenn kein case zutrifft
}
```

## Grundlegendes Beispiel
```java
int tag = 3;

switch (tag) {
    case 1:
        System.out.println("Montag");
        break;
    case 2:
        System.out.println("Dienstag");
        break;
    case 3:
        System.out.println("Mittwoch");
        break;
    case 4:
        System.out.println("Donnerstag");
        break;
    case 5:
        System.out.println("Freitag");
        break;
    default:
        System.out.println("Wochenende");
}
```

**Ausgabe:** `Mittwoch`

## Wichtige Komponenten

### Der switch-Ausdruck
Der Ausdruck in den Klammern wird ausgewertet. Erlaubte Datentypen sind:
- `byte`, `short`, `int`, `char`, `String`

### case-Labels
Jeder case repräsentiert einen möglichen Wert. Die Werte müssen konstant sein (Literale oder final-Variablen).

### break-Statement
Das `break` beendet den switch-Block. Ohne `break` werden auch die folgenden cases ausgeführt (Fall-through).

### default-Case
Der default-Fall wird ausgeführt, wenn kein case-Label zutrifft. Er ist optional, aber empfehlenswert.

## Praktische Anwendungen

### Notenberechnung
```java
char note = 'B';

switch (note) {
    case 'A':
        System.out.println("Sehr gut - 90-100%");
        break;
    case 'B':
        System.out.println("Gut - 80-89%");
        break;
    case 'C':
        System.out.println("Befriedigend - 70-79%");
        break;
    case 'D':
        System.out.println("Ausreichend - 60-69%");
        break;
    default:
        System.out.println("Nicht bestanden");
}
```

### Mehrere Cases für denselben Code
```java
String monat = "Februar";

switch (monat) {
    case "Januar":
    case "März":
    case "Mai":
    case "Juli":
    case "August":
    case "Oktober":
    case "Dezember":
        System.out.println("31 Tage");
        break;
    case "April":
    case "Juni":
    case "September":
    case "November":
        System.out.println("30 Tage");
        break;
    case "Februar":
        System.out.println("28 oder 29 Tage");
        break;
    default:
        System.out.println("Ungültiger Monat");
}
```

### Menüauswahl
```java
int auswahl = 2;

switch (auswahl) {
    case 1:
        System.out.println("Neues Spiel starten");
        break;
    case 2:
        System.out.println("Spiel laden");
        break;
    case 3:
        System.out.println("Einstellungen öffnen");
        break;
    case 4:
        System.out.println("Beenden");
        break;
    default:
        System.out.println("Ungültige Auswahl");
}
```

## Fall-through-Verhalten

Wenn du das `break` weglässt, werden alle nachfolgenden Anweisungen ausgeführt, bis ein `break` oder das Ende des switch-Blocks erreicht wird:
```java
int zahl = 2;

switch (zahl) {
    case 1:
        System.out.println("Eins");
        // Kein break!
    case 2:
        System.out.println("Zwei");
        // Kein break!
    case 3:
        System.out.println("Drei");
        break;
    default:
        System.out.println("Andere Zahl");
}
```

**Ausgabe:**
```
Zwei
Drei
```
## switch vs. if-else

**Verwende switch, wenn:**
- Du einen einzelnen Wert mit mehreren konstanten Werten vergleichst
- Du viele verschiedene Fälle hast (>3)
- Die Übersichtlichkeit wichtig ist

**Verwende if-else, wenn:**
- Du komplexe Bedingungen prüfen musst (z.B. `x > 10 && y < 5`)
- Du Bereiche vergleichen willst (z.B. `x >= 0 && x < 100`)
- Du nicht-konstante Werte vergleichen musst

**Vergleich:**
```java
// Mit if-else
String farbe = "rot";
if (farbe.equals("rot")) {
    System.out.println("Stopp");
} else if (farbe.equals("gelb")) {
    System.out.println("Achtung");
} else if (farbe.equals("grün")) {
    System.out.println("Fahren");
}

// Mit switch (übersichtlicher)
switch (farbe) {
    case "rot":
        System.out.println("Stopp");
        break;
    case "gelb":
        System.out.println("Achtung");
        break;
    case "grün":
        System.out.println("Fahren");
        break;
}
```
